# Training Calendar Framework

This workspace contains two implementation artifacts for the Suunto `Training Calendar` module:

- `docs/training-calendar/interaction-framework.md`: the interaction framework specification in Chinese
- `prototype/training-calendar/index.html`: a local interactive prototype for the main flows

## Verify the logic model

```bash
node --test prototype/training-calendar/model.test.js
```

## Open the prototype

Open `prototype/training-calendar/index.html` in a browser or the Codex in-app browser.
